 # Employee Management System

A web-based Employee Management System developed using PHP and MySQL.

## Features
- Add, view, update, and delete employee records
- Secure form handling
- Responsive user interface

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Bootstrap
- PHP
- MySQL

## How to Run
1. Install XAMPP or WAMP.
2. Place the project folder inside `htdocs`.
3. Import the database using phpMyAdmin.
4. Run the project on `http://localhost/employee-management-system`.
